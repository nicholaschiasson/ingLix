# cranz

The English to inglix transpiler.

## Usage

```
cranz 0.1.0
Command-line utility to transpile a text file between Enlish and inglix, interactively updating a
dictionary on the fly with any as-of-yet untranspiled words.

USAGE:
    cranz [OPTIONS] <FILE>

ARGS:
    <FILE>    File to transpile from English into inglix

OPTIONS:
    -d, --dictionary <DICTIONARY>    File to use as persistant dictionary [default:
                                     ./English_inglix_Dictionary.toml]
    -h, --help                       Print help information
    -V, --version                    Print version information
```
